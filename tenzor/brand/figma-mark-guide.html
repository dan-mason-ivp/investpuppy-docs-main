Before you start: Open your existing Vektor Figma file. Treatment 4b is now locked (May 2026). This guide walks through (A) updating the Vektor maker's mark to the locked Treatment 4b specification, then (B) building the Tenzor mark from scratch. Complete Part A first — it establishes the colour styles and text styles you will reuse for Tenzor.
SETUP File & Style Setup Do this once before anything else
S1 Create or verify colour styles
Why this matters: Colour styles mean every element in both marks updates instantly if a colour ever changes. Do this before drawing anything.
1

In the Figma toolbar, click the Assets panel (or press Shift+I). Go to Libraries → Local styles.

2

Click the + next to Color styles. Create the following styles — use these exact names and hex values:

Brand/Dark#0A0A0AAll mark backgrounds
Mark/Gold#C8A96EVektor V arrowhead; Tenzor Z letter and crossbar

Maker/Grey#AAAAAABY and INVEST in maker's mark
Brand/White#FFFFFFMonochrome variant
3

If these styles already exist in your Vektor file, verify the hex values match exactly. Rename them if necessary to match the names above — consistent naming is essential as Tenzor is added.

S2 Create text styles
1

Select any text layer, then in the right panel click the four-dot grid icon next to the text style field to open Text styles. Click + to create a new style.

2

Create these text styles. Note on letter spacing: Figma expresses letter spacing as a percentage. 0.16em = 16%, 0.20em = 20%, 0.30em = 30%. Enter the % value in the letter spacing field.

3

These styles capture font and tracking only — not colour. Colour is applied separately per instance so the same style can render in Platinum, Gold, Grey, Verdigris, or White depending on context.

PART A Update the Vektor Mark Existing file · maker's mark only
A1 Locate the existing maker's mark text
1

Open your existing Vektor Figma file. In the Layers panel, find the frame or group containing the full Vektor mark.

2

Expand the layer tree and find the text layer that reads BY INVESTPUPPY. It will currently be a single text layer, left-aligned, in a single grey colour.

3

Double-click the text layer on the canvas to enter text editing mode. Confirm it reads exactly BY INVESTPUPPY as one string. Note the current font size — you will use this to calculate the BY size.

A2 Replace the single-line treatment with Treatment 4b
Do not modify the VEKTOR wordmark or V arrowhead. Work only on the BY INVESTPUPPY text layer. If the layer is inside a locked group, unlock it first: right-click the group → Unlock.
1

Delete the existing BY INVESTPUPPY text layer. Take note of its Y position and horizontal centre point before deleting — you will re-create the maker's mark at the same vertical position.

2

Create a new Frame (press F) at the same Y position. This will be the maker's mark container. Enable Auto layout on it: press Shift+A. Set direction to Vertical, alignment to Centre, gap between items to 2.

3

Inside the frame, create the BY text layer: press T, type BY. Apply style Maker/BY. Set font size to 70% of the INVESTPUPPY font size (e.g. if INVESTPUPPY will be 10px, BY is 7px). Set fill colour to style Maker/Grey (#AAAAAA). Set alignment to Centre.

4

Create the INVESTPUPPY text layer below BY in the same frame: press T, type INVESTPUPPY. Apply style Maker/INVESTPUPPY. Set alignment to Centre.

Now select just the characters INVEST within the text layer and set their fill to Maker/Grey (#AAAAAA). Select PUPPY and set fill to Maker/Verdigris (#6BAF82).

5

Select the auto-layout frame. In Alignment, set horizontal alignment to Centre. Resize the frame width to match the width of the VEKTOR wordmark above — this ensures the maker's mark centres correctly beneath the wordmark.

6

Position the maker's mark frame: centre it horizontally to the VEKTOR wordmark group. The vertical position should match where the original BY INVESTPUPPY text sat.

Checking alignment: Select both the VEKTOR wordmark group and the maker's mark frame. Use Align horizontal centres in the right panel to confirm they share the same centre axis.
A3 Group and save as component
1

Select the VEKTOR wordmark group and the maker's mark frame together. Group them: Cmd+G (Mac) / Ctrl+G (Win). Name the group Vektor / Standard.

2

Convert to a component: right-click the group → Create component (or Cmd+Alt+K / Ctrl+Alt+K). This is now the master Vektor mark component.

3

Duplicate the component (Cmd+D) and rename the duplicate Vektor / Mono. On this variant, select all text elements and set colour to Brand/White. This is your monochrome variant — keep it alongside the standard.

PART B Build the Tenzor Mark New mark from scratch · work at 200px wordmark size
Work at 200px font size. Build the mark large — it is much easier to get the crossbar geometry right at large scale. You will create smaller sizes as component instances later, not by redrawing.
B1 Set up a working frame
1

Create a new frame (press F, drag a rectangle). Set fill to Brand/Dark (#0A0A0A). Make it approximately 1200 × 600 — you will trim it later.

2

Name this frame Tenzor — Working. This is your construction area, not the final component.

B2 Create the TENZOR wordmark — letters T, E, N, O, R
1

Press T to create a text layer inside the frame. Type TENZOR.

2

Apply the text style Wordmark/Tenzor (Montserrat Regular, 16% tracking). Set font size to 200.

3

Select the entire text and set fill to Mark/Platinum (#9EA8B3). All six letters are now platinum.

4

Double-click the text layer to enter editing mode. Select only the letter Z (the fourth character). Change its fill to Mark/Gold (#C8A96E). Exit text editing mode.

Check: T, E, N, O, R should be Platinum. Z should be Gold. All letters are the same size.
B3 Create the Z crossbar
This is the most precise step. Take your time. The crossbar is a separate shape layer placed on top of the Z letter — it is not part of the text.
1

Measure the Z letter bounds. Click on the TENZOR text layer. In the right panel, note the X, Y, Width and Height of the entire text layer. Then double-click the text to enter editing mode, select just Z, and note its approximate X position and width. At 200px Montserrat Regular, the Z is approximately 120–130px wide. Exit text editing.

Tip: Place a temporary rectangle over the Z letter to measure its exact bounds. Delete it after.

2

Create the crossbar rectangle. Press R and draw a rectangle. Set its dimensions in the right panel:
— Width: Z_width × 1.16 (Z width plus 8% overhang each side). At ~125px Z width: 145px.
— Height: Font_size × 0.028. At 200px: 5.6px. Round to 6px for production at this scale.

3

Set the rectangle fill to Mark/Gold (#C8A96E). Remove any border/stroke.

4

Round the ends. In the right panel under corner radius, set all four corners to 3 (half the height, creating a pill shape). At 6px height, set corner radius to 3.

5

Position the crossbar. Place the rectangle centred horizontally on the Z letter. Vertically, position it at 52% of the Z cap-height from the baseline. At 200px Montserrat, the cap-height is approximately 144px — so 52% = ~75px from the baseline of the text layer. Adjust by eye until it sits just above the midpoint of the Z.

6

Rotate −15°. With the rectangle selected, in the right panel find the rotation field (the circular arrow icon). Enter -15. The crossbar should now tilt upward from left to right across the Z.

7

Fine-tune position. After rotation the crossbar will shift slightly. Re-centre it horizontally on the Z and confirm it still crosses at approximately the mid-height of the letter. Use the canvas zoom (Z) to get a close view.

Visual check: The crossbar should create a clean diagonal accent across the Z. It should feel like the same visual weight as the Vektor V arrowhead — an accent, not a dominant element. If it looks too heavy, reduce the height slightly.
B4 Group the wordmark
1

Select both the TENZOR text layer and the crossbar rectangle. Group them: Cmd+G / Ctrl+G. Name the group TENZOR wordmark.

2

Confirm the group bounds are tight around the wordmark. If the group has excess whitespace, select it and use Resize to fit or manually trim the bounding box.

B5 Add the maker's mark (Treatment 4b)
This is identical to the process in Part A, Step A2. You are creating the same maker's mark structure. If you have already done Part A, you can duplicate the maker's mark frame from the Vektor mark and reuse it here — the structure is identical.
1

Create a new frame below the wordmark group. Press F, draw the frame. Enable auto layout (Shift+A), set to Vertical, alignment Centre, gap 2.

2

Inside the frame, create the BY text layer: type BY, apply style Maker/BY. Font size = 13px (70% of 19px INVESTPUPPY reference at this scale — adjust proportionally if your INVESTPUPPY size differs). Colour: Maker/Grey. Alignment: Centre.

3

Create the INVESTPUPPY layer: type INVESTPUPPY, apply style Maker/INVESTPUPPY. Font size = 19px (approximately 9.5% of 200px wordmark cap-height). Alignment: Centre.

Select INVEST → colour Maker/Grey. Select PUPPY → colour Maker/Verdigris.

4

Set the auto-layout frame width to match the wordmark group width. Centre-align horizontally to the wordmark group. Position vertically: gap between bottom of wordmark and top of BY line = approximately 18% of the wordmark font size = 36px at 200px scale.

B6 Create the full mark component and variants
1

Select both the TENZOR wordmark group and the maker's mark frame. Group them (Cmd+G). Name the group Tenzor / A — Full mark. Convert to component: right-click → Create component.

2

Create Variant B (clean, no crossbar). Duplicate the component (Cmd+D). Rename it Tenzor / B — Clean. In this duplicate, find the crossbar rectangle inside the TENZOR wordmark group and delete it. The Z letter retains its gold colour — the only change is removing the crossbar.

3

Create Variant C (monochrome). Duplicate Variant B. Rename it Tenzor / C — Mono white. Select all colour-bearing elements and set every fill to Brand/White (#FFFFFF): the wordmark letters (T, E, N, Z, O, R), BY, INVEST, and PUPPY. Z retains no crossbar and becomes white. PUPPY becomes white (colour split not used in monochrome).

4

Create Variant C black (reversed). Duplicate Variant C. Rename it Tenzor / C — Mono black. Set all text fills to #000000. This variant is for use on white/light backgrounds only.

B7 Use Figma component variants to manage all sizes
Do not redraw the mark at smaller sizes. Place component instances and scale them — the proportions hold because everything is built from text styles and relative values. The crossbar will visually thin at smaller sizes because its height formula is proportional to font size.
1

To place a smaller instance: from the Assets panel, drag a Tenzor / A — Full mark component onto the canvas. In the right panel, set the width to your target size while holding Shift to maintain proportions.

2

At or below 18px wordmark size — switch to Variant B. Click the placed instance, then in the right panel under Instance, swap it to Tenzor / B — Clean.

3

Create a dedicated scale reference page in the file: add a page called Scale Reference. Place instances of Variant A at each standard size (96px, 72px, 48px, 32px, 22px, 18px) and Variant B at smaller sizes (14px, 10px, 7px) on a dark background frame. This gives you and any collaborators an at-a-glance reference.

PART C Exporting Assets SVG · PNG · PDF
C1 Set up export settings on each component
1

Select the master component (e.g. Tenzor / A — Full mark). In the right panel scroll to Export. Click + to add export settings.

2

Add the following export settings for each component:

FormatScale / SizeSuffix
PNG1×(blank)
PNG2×@2x
PNG4×@4x
SVG1×(blank)
PDF1×(blank)

3

Repeat for all four Tenzor variants and both Vektor variants.

C2 Export
1

Select all mark components on the canvas. In the right panel, click Export at the bottom. Figma will export all selected layers with all configured export settings in one batch.

2

Files will be named by layer name + suffix. The layer names Tenzor / A — Full mark etc. will produce filenames like Tenzor / A — Full mark@2x.png. Rename to the convention in the brand spec: tenzor-A-2x.png etc.

3

SVG check before delivery: Open each exported SVG in a text editor and confirm there are no embedded fonts (the text should be converted to outlines or the correct font should be referenced). If fonts are embedded, in Figma select all text in the mark, right-click → Outline stroke to convert text to paths before exporting SVG.

Convert text to outlines for production SVG. For any SVG that will be used in print, embroidery, signage, or any context where Montserrat/Poppins may not be available, text must be outlined first. Keep the editable (live text) version as the master in Figma and outline a copy for each production SVG export.
